# Danan the Airplane

Ett enkelt webbspel där flygplanet **Danan** flyger fram och hoppar över hinder.

- **Space / ↑ / Tap** — hoppa
- **P** — paus
- **R** — omstart
- Varje **20 m** = 1 poäng, och spelet blir snabbare för varje poäng.

Spelet körs helt i webbläsaren (vanilla HTML/CSS/Canvas) och hostas via GitHub Pages.

## Köra lokalt

Öppna `index.html` direkt i en webbläsare, eller starta en lokal server:

```bash
python -m http.server 8000
```

och gå till http://localhost:8000.
